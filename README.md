LiveSplit.AVP2010
=================

This is a [LiveSplit] component for Aliens vs. Predator (2010)

Features
--------

  * Removes load times.

Requirements
------------

  * Aliens vs. Predator (2010) - DirectX 11 only!!
  * LiveSplit 1.4+
  * .NET Framework 4
